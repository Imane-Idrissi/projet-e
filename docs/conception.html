6. Règles de Gestion

Récapitulatif des règles métier implémentées dans le module.

CodeRègleImplémentationCouche
RG-PRD-01 Référence unique auto-générée au format PRD-XXXXX ProduitService.genererReference() Service
RG-PRD-02 Le prix HT doit être strictement positif @DecimalMin("0.01") sur CreateProduitRequest DTO / Validation
RG-PRD-03 Les catégories sont hiérarchiques (parent/enfant) Relation @ManyToOne récursive sur Categorie Entité
RG-PRD-04 Un produit est archivé (soft delete), jamais supprimé ProduitService.archiverProduit() → actif = false Service
RG-PRD-05 Alerte quand quantiteStock ≤ seuilAlerte Produit.isStockBas() + findProduitsStockBas() Entité / Repository
